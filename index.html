Gazetteer

Gazetteer is a Single Page Page that uses LeafletJS. It also contains APIS for Currency Conversion, local weather, Covid Statistics, News and WebCam. Users navigating to Gazetteer will be greeted with a pre loader giving the webpage time to load so that the user does not get a weak impression of the webpage. Users have the ability to use their device in-built Navigation to locate the country they are in for a quick info. Html and css were used mainly for the styling of the project. JavaScript was used mainly for creating functions and interacting with the user for a smoother user experience. I did depend on the power of AJAX for this to be possible, ensuring proper event loop sequences. Php was used mainly for handling API request which returned the accurate information from the API in a timely manner. The data returned from the API in PHP was then passed on to the calling Javascript AJAX and finally Javascript was able to display the data to HTML using JQUERY or futher pass the data to another AJAX function.

Company Directory

Company Directory is a Single Page Page that showscase a popluar functionality of any Company. It displays the Employees, Departments and Branches of a company.

Company Directory resolves around the idea of CRUD, giving the user the ability to either create, edit, update and delete data from the DataBase without the need for SQL knowledge. Company Directory is designed to be easy for all range of users to navigate and interact. Users are first greeted with pre-loader for a smooth user experience. I tried to imimate the look and feel of a phone contact that all users are sure to be familiar with. HTML and CSS are used mainly for the styling of Company Directory. Font-awesome CDN is utilized for their free icons. Details are arranged Alphabetically based on their first name. Every button that users click is greeted with a beautiful popup. Sweet Alert is used instead of the conventional popups/alert for its versatile appearance and ability. Either creating, reading, updating or deleting, Javascript is run in the background to use the power of AJAX to interact with events asynchronously. PHP is used to write query to MYSql DATABASE, updating the data in database right from the browser. To ensure that datas that have dependency are not deleted, I have added a script that checks for dependency. Users are able to see the changes that they have made easily.
